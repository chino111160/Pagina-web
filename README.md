## 🚀 Cómo visualizar el sitio

1. Clona el repositorio:
```bash
   git clone https://github.com/chino111160/Pagina-web.git
```
2. Abre el archivo `index.html` en tu navegador web.

O bien, publica el sitio gratuitamente usando **GitHub Pages**:
- Ve a *Settings → Pages → Branch: main → Save*
- Tu sitio estará disponible en: `https://chino111160.github.io/Pagina-web/`
